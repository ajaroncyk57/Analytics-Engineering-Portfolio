# Metric Definitions

## Average Basket Size
Average Order Value (AOV) during the relevant period.

Formula:
```text
Average Basket Size = Total Sales / Order Count
```

## Weekly Visits per Loyalty Member
```text
Weekly Visits per Loyalty Member = Total Loyalty Visits / Loyalty Member Count
```

## Coffee Attach Rate
```text
Coffee Attach Rate = Coffee Orders / Total Orders
```

## Cross-Sell Lift
```text
Cross-Sell Lift = Treatment Non-Coffee Change - Control Non-Coffee Change
```
